개인정보 수정

수정하고자 하는 개인정보의 ID로 개인정보를 수정 할 수 있습니다. 수정 하고자 하는 컬럼명과 데이터를 jsonString 형태로 요청해야 합니다.
주의사항
ID, PASSWORD, SALT는 수정 불가

  • 요청 Parameters

    • id

      필수사용자 id

      jsonString

      필수string

      사용자가 생성한 테이블 컬럼과 해당되는 데이터를 json 형태로 나열한 문자

Response

{
  "error": "error message",
  "success": true
}
