회원가입

사용자가 등록한 테이블의 컬럼에 맞는 양식에 따라 회원가입을 진행 할 수 있습니다. 사용자가 필요로 하는 데이터에 맞는 컬럼을 테이블에 추가하면 사용 할 수 있습니다.
주의사항
kokonut 화면에서 설정한 테이블의 컬럼명과 타입에 맞게 작성되어야 합니다.

  • 요청 Parameters

    • jsonString

      필수json (그룹화 id, pw)

      사용자가 생성한 테이블 컬럼과 해당되는 데이터를 json 형태로 나열한 문자열

Response

{
  "error": "error message",
  "success": true
}
