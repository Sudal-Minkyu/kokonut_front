ID 찾기

등록한 개인정보의 ID를 잊었다면 개인정보 테이블에서 추가한 컬럼 중 암호화 되어있는 컬럼을 이용하여 ID를 찾을 수 있습니다. 개인정보 테이블의 컬럼명과 타입에 맞게 하나의 필드로 요청을 하면 해당 하는 개인정보의 ID를 반환합니다.
주의사항
주의사항 : 개인정보 테이블에 ‘ID’라는 컬럼이 존재 해야하며, 한 개 이상의 암호화 속성을 가진 컬럼을 사용하여야 합니다. 2가지 조건이 충족되지 않으면 해당 API를 사용 할 수 없습니다.

  • 요청 Parameters

    • jsonString

      필수string

      사용자가 생성한 테이블 컬럼과 해당되는 데이터를 json 형태로 나열한 문자열

Response

{
  "error": "error message",
  "id": "id",
  "success": true
}
